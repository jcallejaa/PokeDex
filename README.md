# PokeDex
----------
A simple "Pokedex" consuming data in https://pokeapi.co and using:

- Kotlin<br>
- CleanCode<br>
- MVP <br>
- Fragments<br>
- ConstraintLayout<br>
- RecycerView <br>
- CoordinatorLayout<br>
- Coroutines <br>
- Unit tests <br>
- SearchBar <br>

And:
- Glide (Loading Images in background) <br>
- Lottie (Show vector animations)<br>
